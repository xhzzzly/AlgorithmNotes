# 二叉索引树

### 区间查询

```cpp
int query(int x, int BIT[]) {
    int sum = 0;
    for (; x > 0; x -= x & -x)
        sum += BIT[x];
    return sum;
}
```

### 单点修改

```cpp
void update(int x, int delta, int BIT[], int n) {
    for (; x <= n; x += x & -x)
        BIT[x] += delta;
}
```

### 复杂度

空间为 
$$
\mathcal{O}(n)
$$
两种操作的时间复杂度都为
$$
\mathcal{O}(\log_2n)
$$
